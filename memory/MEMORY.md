# Memory — site-nyres

## Projeto
Landing page para Nyres Edim, terapeuta Pathwork® com +25 anos de experiência. Atendimento individual e grupos de estudos em BH e online. Objetivo: converter visitantes em agendamentos via WhatsApp (+55 31 98585-0589).

## Público-alvo
Pessoas que já compreendem seus padrões emocionais mas continuam repetindo ciclos. Buscam transformação profunda, não terapia convencional. Dores: falta de autovalorização, padrões repetitivos, desalinhamento com propósito, ansiedade, culpa, cansaço profundo.
